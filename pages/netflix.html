Netflix Data Analysis

Exploratory data analysis on Netflix's content catalog (~8800 titles) — identifying distribution patterns, genre concentration, temporal trends, and geographic production insights using Pandas and Matplotlib.

Overview

Objective

Analyze Netflix's content library to understand: what types of content dominate the catalog, how content additions have changed over time, which countries produce the most content, and whether there are measurable patterns in genre distribution across regions.

Dataset

Netflix Movies and TV Shows dataset from Kaggle (~8807 titles) with 12 columns: show_id, type (Movie/TV Show), title, director, cast, country, date_added, release_year, rating, duration, listed_in (genres), and description.

Approach

Standard EDA workflow: data cleaning (missing value handling, type conversion, multi-value column parsing) → univariate analysis → bivariate/cross-tabulation → temporal trend analysis → visualization. All analysis in Jupyter Notebook.

Analysis Architecture

The analysis follows a structured question-driven approach. Each visualization answers a specific analytical question about the dataset — no exploratory charts without purpose.

Data Cleaning Layer

Missing value handling: director (~30% null), cast (~10% null), country (~7% null), date_added (~0.1% null). Strategy: dropped rows with missing date_added (needed for temporal analysis), filled director/cast with "Unknown" for non-critical analysis, dropped missing country for geographic analysis.

Feature Engineering

Extracted year and month from date_added for temporal grouping. Split multi-value columns (listed_in, country, cast) on comma delimiter for per-genre and per-country aggregation. Created duration_minutes (for movies) and n_seasons (for TV shows) as separate numeric columns.

Aggregation & Grouping

GroupBy operations on type, country, genre, rating, and year_added. Cross-tabulations for country×genre and type×rating combinations. Value counts, pivot tables, and rolling averages for trend analysis.

Analysis Pipeline

1

Data Ingestion & Cleaning

Loaded CSV (~8807 rows × 12 columns). Converted date_added to datetime. Identified and handled missing values (30% missing directors, 10% cast, 7% country). Removed 10 rows with missing date_added. Verified no duplicate show_ids.

2

Content Distribution Analysis

Analyzed Movies vs TV Shows split: ~69.6% Movies, ~30.4% TV Shows. Explored content ratings distribution — TV-MA (36%), TV-14 (24%), TV-PG (11%) are the top 3 ratings. Netflix's catalog skews heavily toward mature/adult content.

3

Temporal Trend Analysis

Plotted content additions by year (2008–2021). Identified exponential growth phase (2015–2019) with peak additions in 2019 (~2000 titles added). Post-2019 slowdown consistent with Netflix's shift from licensing to original production.

4

Geographic Analysis

Identified top 10 producing countries by title count. US leads (~2800 titles), followed by India (~970), UK (~420), Japan (~240), South Korea (~200). Analyzed country-specific genre distributions using cross-tabulation.

5

Genre & Duration Analysis

Top genres by frequency: International Movies, Dramas, Comedies, Action & Adventure, Documentaries. Average movie duration: ~100 minutes. Most TV shows are 1 season (~72%), indicating high volume of limited series or shows not renewed.

6

Visualization & Reporting

Created 12+ charts: bar charts (content type, ratings, top countries), line plots (additions over time), heatmap (country × genre), pie chart (type split), histogram (duration distribution). All charts annotated with titles, labels, and contextual notes.

Content Growth Peaked in 2019

Content additions grew exponentially from 2015–2019, then declined. This aligns with Netflix's public shift from bulk licensing third-party content to investing in original production — which has a longer production pipeline and slower release cadence.

India's Rising Share

India's contribution grew significantly from 2018 onward, reflecting Netflix's investment in regional content markets. Indian content is heavily concentrated in "International Movies" and "Dramas" genres.

Genre Concentration

"International Movies" and "Dramas" are the most tagged genres across nearly all countries — but these are broad, often co-applied labels. Comedies rank third globally but are proportionally higher in US-produced content. Anime is disproportionately represented in Japanese content.

Takeaway

EDA Fundamentals

Practice with the full exploratory data analysis workflow: cleaning messy real-world data, handling missing values with different strategies per column, extracting features from complex multi-value fields, and creating purposeful visualizations.

Pandas Proficiency

Extensive use of Pandas for data manipulation: string splitting, datetime conversion, groupby aggregations, cross-tabulation, value counts, pivot tables, and multi-index operations — all on a real-world dataset with genuine data quality issues.

Data Storytelling

Each chart answers a specific question. The analysis follows a logical progression — from broad distribution overview to temporal trends to geographic deep-dives — demonstrating structured analytical thinking rather than random chart generation.

Challenges & Learnings

Missing Data Strategy

30% missing directors required a decision: drop rows (lose data), fill with "Unknown" (lose accuracy), or exclude director from analysis. Used different strategies per analysis type — dropped for director-specific analysis, kept for genre/temporal analysis.

Multi-Value Column Parsing

Genres, countries, and cast are stored as comma-separated strings. Splitting and exploding these for per-genre or per-country analysis inflates row counts. Required careful groupby logic to avoid double-counting titles.

Genre Label Noise

"International Movies" is applied alongside specific genres (Drama, Thriller, etc.), inflating its frequency. Similarly, "TV Dramas" and "Dramas" are separate labels in the dataset despite conceptual overlap. Required manual label-aware analysis.

Temporal Bias

Date_added reflects when Netflix added the title, not when it was produced. A 1990s movie added in 2020 skews temporal analysis. Separated release_year (production) from date_added (catalog addition) to avoid conflating the two.

Country Multi-Production

~12% of titles list multiple countries (e.g., "United States, United Kingdom"). These co-productions inflate counts for both countries. Kept both attributions with a note on double-counting in the analysis.

Visualization Design

Default Matplotlib styling produces generic-looking charts. Invested time in consistent color palettes, axis formatting, annotation placement, and figure sizing to make visualizations presentation-ready rather than default-output quality.

Limitations

No Viewership Data

The dataset contains catalog metadata only — no watch counts, completion rates, engagement metrics, or user ratings. All insights describe what Netflix has, not what users actually watch. Content popularity cannot be assessed.

Static Snapshot

The dataset is a point-in-time snapshot (likely mid-2021). Netflix continuously adds and removes content. Findings may not reflect the current catalog, especially given Netflix's post-2021 content strategy changes.

Descriptive Only

This is purely exploratory analysis — no predictive modeling, no hypothesis testing, no causal claims. The findings describe patterns in the data and suggest possible explanations, but do not constitute business recommendations.

No External Validation

Findings were not cross-referenced with Netflix's actual public reports or investor data. Aggregate numbers (content counts, growth rates) could be verified against Netflix's published data but were not in this analysis.
